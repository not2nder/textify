# ✨ Textify

Converta qualquer texto em fontes estilizadas de forma rápida e simples.

## 📖 Sobre o projeto

O **Textify** é uma aplicação web que transforma textos comuns em diferentes estilos de fontes utilizando caracteres Unicode. Ideal para uso em redes sociais, bios, nomes personalizados e qualquer lugar onde você queira destacar seu texto. O projeto é uma versão mais moderna e atualizada do meu antigo projeto: [Fancy Words](https://not2nder.github.io/fancy-words/)

---

## 🛠️ Tecnologias utilizadas

- React
- Vite
- Tailwind CSS
