# Textify | Fontes Estilizadas

Converta qualquer texto em fontes estilizadas de forma rápida e simples.

## Sobre o projeto

Transforme textos comuns em diferentes estilos de fontes. Ideal para uso em redes sociais, bios e nicknames personalizados. O projeto é uma versão mais moderna e atualizada do meu antigo projeto: [Fancy Words](https://not2nder.github.io/fancy-words/)

## Tecnologias utilizadas

- Vite + React
- Tailwind CSS
